Configuration Supabase

1. Crée le projet Supabase

Va sur supabase.com, crée un projet, puis renseigne tes clés en haut du script (SUPABASE_URL et SUPABASE_ANON_KEY).

2. Crée la table levels

Dans l'éditeur SQL de Supabase, exécute :

create table levels ( id bigint generated always as identity primary key, name text not null, top_rank text, description text, best_run text default '0%', progress integer default 0, runs_from_0 text default '0 → 100%', attempts text default '0', worst_fail text, enjoyment numeric(3,1), image_url text, status text default 'pas_commence', difficulty text default 'Extreme Demon', created_at timestamptz default now() ); -- Si la table existe déjà, ajoute les nouvelles colonnes : -- alter table levels add column worst_fail text; -- alter table levels add column enjoyment numeric(3,1); -- Active le temps réel alter publication supabase_realtime add table levels;

3. Crée ton compte admin

Dans Supabase → Authentication → Users → Invite user, crée ton compte email/mot de passe. C'est celui que tu utiliseras pour te connecter ici.

4. Active le Realtime

Supabase → Database → Replication → active la table levels. Les cartes se mettront à jour automatiquement pour tous les viewers !
